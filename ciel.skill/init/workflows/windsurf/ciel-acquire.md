# Skill Acquisition

## Steps

1. **Need Analysis** — Define capability gap, check existing skills
2. **Registry Tier** — Search ~/.ciel/skills/ for match
3. **MCP Tier** — Check configured MCP servers
4. **Web Tier** — Research available implementations
5. **Integration** — Harmonize with existing skills
