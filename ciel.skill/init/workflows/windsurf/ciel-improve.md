# Self-Improvement

## Trigger

Growth signal detected from interaction outcome.

## Steps

1. **Signal Analysis** — Review trajectory for lessons
2. **Proposal Generation** — Draft improvement (config, prompt, or skill)
3. **Council Gate** — Present to Council if non-trivial
4. **Apply** — Execute improvement, git commit
5. **Observe** — Monitor subsequent interactions
